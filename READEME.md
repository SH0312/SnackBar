Snack Bar API server

이용자 구분
- 사용자
- 관리자

기능
- 사용자
  - 로그인
  - 상품 주문
  - 상품 주문 취소
  - 상품 주문 상태 - 대기, 준비중, 준비 완료
  - 상품 주문이력 - 주별 월별 조회
- 관리자
  - 매점 ON/OFF
  - 매점 ON/OFF 상태 보기
  - 관리자 계정 관리
    - 관리자 추가
    - 관리자 해제
  - 상품 주문 목록
  - 상품 주문 상태 변경
    - 취소
    - 준비중
    - 준비완료
  - 상품관리
    - 등록
    - 수정
    - 삭제
- 공통
  - 등록된 상품 조회
  - 주문 웨이팅 조회
